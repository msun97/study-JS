연산자

할당 연산자

변수에 값을 할당할 때 사용합니다. =, +=, -=, *=, /=, %=, **=

교대 할당 연산자

변수를 특정 방향으로 이동한다. <<=, >>=, >>>=

비트 단위 할당 연산자

비트 단위 AND, OR, XOR 연산을 수행하고 결과를 변수에 할당한다. &=, ^=, |=

논리 할당 연산자

&&= : 첫번째 값이 참이면 두번째 값이 할당 ||=(or), ??=(NULL)

산술 연산자

산술 되어지는 두 변수를 피연사자라고 한다.

덧셈 연산자

변수에 값을 더할 때 사용합니다. 문자열도 가능합니다. +, -

곱셈 연산자

변수에 값을 곱할 때 사용합니다. 문자열도 가능합니다. *, /, %

% 연산자는 나눈 후 나머지를 반환 합니다.

증가 연산자

변수에 1을 더할 때 사용합니다. ++, --

지수화

변수를 제곱할 때 사용합니다. **

이는 Math.pow() 메서드로도 가능합니다.

비교 연산자

변수의 값이 같은지 비교합니다. 문자열도 비교할 수 있습니다. >, <, >=, <=, ==, ===,!=,!==

논리 연산자

두개의 boolean 값(true, false)을 연산합니다. &&(AND), ||(OR),!(NOT)

유형 연산자

변수의 데이터 유형을 확인합니다. typeof

비트 연산자

변수의 비트를 연산합니다. &, |, ^, ~, <<, >>, >>>
