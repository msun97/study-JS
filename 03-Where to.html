Script 태그

HTML <script> 태그를 이용하여 JavaScript 코드를 HTML 문서 내에 삽입할 수 있습니다.

함수 및 이벤트

JavaScript 함수, 이벤트, DOM API 등을 이용하여 HTML 문서 내에서 JavaScript 로직을 작성할 수 있습니다. 예를 들어, 사용자가 버튼을 클릭하는 등의 이벤트가 발생 하면 함수가 호출 될 수 있습니다.

scirpt태그나 JS 외부 파일을 불러오는 것은 head, body 태그 어디서나 가능하다. 외부 스크립트는 scrip에 src 속성을 추가한다.

외부 JavaScript의 장점

외부 JavaScript 파일을 사용하면 HTML, CSS, JS 파일이 분리되어, 코드를 쉽게 관리할 수 있습니다. 또, CSS, JS 파일을 분리할 경우, CSS, JS 파일을 다운로드 받아야 하므로, HTML, CSS, JS 파일을 로드하는 속도가 빨라질 수 있습니다.
