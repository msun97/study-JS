변수

자동 선언

선언 되지 않은 변수에 할당할 시 자동으로 변수가 선언된다.

var

var 키워드로 변수를 선언 할 수 있다. var은 현재 사용하지 않는다.

let

let 키워드로 변수를 선언할 수 있다. let은 block scope(함수, if-else, for-loop) 내에서만 유효하다.

const

const 키워드로 변수를 선언할 수 있다. const는 block scope(함수, if-else, for-loop) 내에서만 유효하며, 값을 변경할 수 없다.

식별자

식별자(변수, 함수, 객체, 클래스)는 유일해야 하며, 대소문자를 구분한다. 문자로 시작해야 하며, 숫자, 밑줄, 달러 기호를 포함할 수 있다.

할당 연산자

변수에 값을 할당할 때, '='(할당) 연산자를 사용한다.

데이터 유형

변수에는 JavaScript의 다양한 데이터 유형(string, number, boolean, object, null, undefined)을 저장할 수 있다.

undefinced

변수가 선언되었지만, 초기화(할당)되지 않은 상태일 때, undefined 값을 갖는다.
