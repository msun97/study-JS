Let

블록 범위

let과 const는 블록 범위를 제공한다. 블록 내부에서 선언된 변수는 외부에서 접근할 수 없다.

글로벌 범위

var은 글로벌 범위를 갖는다. var 변수는 전역 스코프(global scope)에서 접근할 수 있다.

재선언

let 변수는 다시 선언 할 수 없다.

    var x = 10;
    {
      var x = 2;
    }

    x=2가 된다.
  
    let y = 10;
    {
      let y = 2;
      console.log(y); // 2
    }

    y=10이다.

호스팅

var 변수는 호스트(hoisting)되어 undefined로 초기화된다. let, const는 호스트되지 않는다. 이때 let 변수를 선언하기 전에 사용하면 레퍼런스 에러가 발생한다.
