Object

객체는 어떠한 변수로, 동일한 속성을 갖지만 값은 다른 대상의 모임이다.

객체 정의

객체 리터럴

객체 리터럴(Object Literal)은 {}를 사용하여 정의한다.
{ name: 'John', age: 30, isStudent: true }

객체 생성

변수 지정하듯 cosnt varsName = {value: key, value2: key2 ...} 로 생성할 수 있다. 이때 varsName과 뒤 객체는 동일한 메모리 주소를 공유한다.

Object() 생성자

Object() 생성자를 사용하여 객체를 생성할 수 있다. 

      const obj = new Object();
      obj.name = 'John';
      obj.age = 30;
      obj.isStudent = true;

객체 유형 생성자

      function objName(property1, property2, ...) {
        this.property1 = property1;
        this.property2 = property2;
        //...
      }

      const obj = new objName(key1, key2, ...);
      const obj2 = new objName(key1, key2, ...);
      ...

이때 속성에 주어진 값은 모든 객체의 기본값이다. 객체 생성자(objName)에는 새 속성을 추가 할 수 없다. 추가하려면 objName.prototype,newProperty="key" 로 추가 해야한다.

객체 속성

객체는 속성(property)과 값(value)으로 구성된다.

접근

객체의 속성(property)에 접근할 때, '.'(dot) 또는 '[]'(bracket) 연산자를 사용한다.
obj.name, obj['name']

객체 메서드

객체는 메서드(method)를 포함할 수 있다. 이때 메서드는 어떤 작업을 수행할 수 있는 방식이다. 메서드는 함수로 저장된다.
obj.sayHello()

메서드 접근

obj.sayHello(), obj['sayHello']()

괄호 없이 접근하면 함수 정의가 반환된다

메서드 추가

obj.sayHello = function() { console.log('Hello'); }

객체 표시

객체 출력

console.log(obj); // [object Object]

객체 속성 표시

obj.property로 표시

루프에서

for (let x in person) { text += person[x] + " "; }; // 각 obj의 key값 출력

Object.value

Object.values(obj).forEach(value => console.log(value)); // obj의 value값 출력

object.entries

Object.entries(obj).forEach(entry => console.log(entry)); // obj의 key-value 목록 출력

JSON.stringify()

JSON.stringify(obj); // JSON 형식으로 obj 출력
