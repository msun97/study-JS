JS | Statement

JS의 Statement은 값, 연산자, 표현식, 키워드 및 주석으로 구성됩니다. 구문은 쓰여진 순서대로 실행됩니다.

세미콜론

세미콜론은 구문을 구분합니다. 각 실행문의 끝에 세미콜론을 추가합니다. 세미콜론으로 구분하면 한 줄에 여러 문장을 입력할 수 있습니다.

공백

JS는 여러 개의 공백을 무시합니다. 스크립트에 공백을 추가하여 더 읽기 쉽게 만들 수 있습니다. 연산자 주위에 공백을 넣으면 좋습니다.

줄 바꿈

대체로 80자가 넘는 코드 줄은 피하고 싶습니다. 연산자 뒤에서 줄을 바꾸는 것이 좋습니다.

코드 블록

JS의 명령문은 중괄호 {...} 안에 코드 블록으로 그룹화할 수 있습니다. 보통 함께 실행되는 명령문을 정의할 때 쓰입니다.

키워드

JS에는 21개의 키워드가 있습니다. 키워드는 JS에서 사용하는 단어로, JavaScript의 기능을 제어하거나 변수, 함수, 상수, 제어문, 반복문, etc. 등을 정의합니다.

  • var : 변수 선언
  • const : 상수 선언
  • let : let 변수 선언
  • function : 함수 선언
  • return : 함수의 반환
  • if : 조건 분기
  • switch : case에 실행 될 명령문 블록 표시
  • for : 루프문
  • try : 오류 처리
