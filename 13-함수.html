함수

구문

function functionName(parameter1, parameter2, ...) { // 함수 구문 }

호출

functionName(argument1, argument2,...) // () 연산자로 호출

이벤트가 발생할 때, JS에서 호출될 때, 자동으로 등 호출된다.

반환

함수는 return에 도달하면 실행이 중지되고 return값을 반환한다.

변수값 함수

함수 역시 변수를 사용하는 것과 같은 방식으로 사용할 수 있다.

지역 변수

함수 내에서 선언된 변수는 해당 함수 내에서만 유효하다. 함수 외부에서도 이 변수를 참조할 수 없다. 지역 변수는 함수가 시작될 때 생성되고, 함수가 완료되면 삭제된다.
